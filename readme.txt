O project 0 (projeto 0) consiste em um clone da página de pesquisa do Google, pesquisa de imagens do Google e pesquisa avançada do Google.
Foi criado apenas com HTML e CSS.
O botão de "estou com sorte", da página de pesquisa do Google, pedirá permissão ao usuário para direciona-lo a uma nova página.
Ainda estou procurando uma forma de consertar isso. Porém, o Youtube e o Google Tradutor não pedirão permissão de redirecionamento.
